CommandKit provides first-class JSX support for both TypeScript and
JavaScript projects. This allows you to write Discord message
components using a familiar and intuitive syntax if you're already
familiar with React.

## Setup

CommandKit automatically configures JSX support in your project. If
you are doing a manual setup, ensure you have the following in your
configuration:

<Tabs>
  <TabItem value="ts" label="TypeScript" default>
    ```json title="tsconfig.json"
    {
      "compilerOptions": {
        "jsx": "react-jsx",
        "jsxImportSource": "commandkit"
      }
    }
    ```
  </TabItem>
  <TabItem value="js" label="JavaScript">
    ```json title="jsconfig.json"
    {
      "compilerOptions": {
        "jsx": "react-jsx",
        "jsxImportSource": "commandkit"
      }
    }
    ```
  </TabItem>
</Tabs>

CommandKit provides JSX versions for the following Discord.js
builders:

- [`ActionRow`](./02-interactive-components/01-action-row.mdx)
- [`Button`](./02-interactive-components/02-button.mdx)
- [`SelectMenu`](./02-interactive-components/03-select-menu.mdx)
- [`Modal`](./02-interactive-components/04-modal.mdx)

## Example usage

<Tabs>
  <TabItem value="ts" label="TypeScript" default>
    ```tsx title="example.tsx"
    import { Container, TextDisplay, ActionRow, Button } from 'commandkit';
    import { ButtonStyle } from 'discord.js';

    const message = (
      <Container>
        <TextDisplay>Welcome to our server!</TextDisplay>
        <ActionRow>
          <Button style={ButtonStyle.Primary}>Click me!</Button>
        </ActionRow>
      </Container>
    );

    interaction.reply({
      components: [message],
    });
    ```

  </TabItem>
  <TabItem value="js" label="JavaScript">
    ```jsx title="example.jsx"
    import { Container, TextDisplay, ActionRow, Button } from 'commandkit';
    import { ButtonStyle } from 'discord.js';

    const message = (
      <Container>
        <TextDisplay>Welcome to our server!</TextDisplay>
        <ActionRow>
          <Button style={ButtonStyle.Primary}>Click me!</Button>
        </ActionRow>
      </Container>
    );

    interaction.reply({
      components: [message],
    });
    ```

  </TabItem>
</Tabs>

## JSX fragments

CommandKit fully supports JSX fragments for grouping multiple elements
without adding extra nodes:

<Tabs>
  <TabItem value="ts" label="TypeScript" default>
    ```tsx title="fragments.tsx"
    const elements = (
      <>
        <TextDisplay>First message</TextDisplay>
        <TextDisplay>Second message</TextDisplay>
      </>
    );

    interaction.reply({
      components: elements,
    });
    ```

  </TabItem>
  <TabItem value="js" label="JavaScript">
    ```jsx title="fragments.jsx"
    const elements = (
      <>
        <TextDisplay>First message</TextDisplay>
        <TextDisplay>Second message</TextDisplay>
      </>
    );

    interaction.reply({
      components: elements,
    });
    ```

  </TabItem>
</Tabs>
